# Start here (Finder)

Den här mappen innehåller bara genvägar (symlinks) för att du snabbt ska hitta rätt del av projektet i Finder – inga filer har flyttats.

## Genvägar
- `01_iOS_App_(Helper)` → iOS-appen (Xcode-projektet ligger i `Helper/Helper.xcodeproj`)
- `02_Backend_(HelpersHelp)` → backend (FastAPI)
- `03_ARKITEKTUR.md` → systemöverblick
- `04_ROOT_README.md` → README i rotmappen

## Tips
- Lägg `00_START_HERE` i Finder-favoriter (dra den till sidofältet).
- Finder → Go → Go to Folder… (⇧⌘G) och klistra in rotmappen om du tappar bort den.
